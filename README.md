# HOW TO USE

git-bash on windows

```
winpty bash
```

build

```
./build.sh
```

encrypt

```
./enc.sh /path/to/source/folder /path/to/destination/folder
```

decrypt (tar)

```
./dec.sh ./dest/20220101_000000.dat /path/to/destination/folder
```

decrypt (decompress)

```
./dec.sh -d ./dest/20220101_000000.dat /path/to/destination/folder
```
