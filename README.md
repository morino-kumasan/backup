# HOW TO USE

git-bash on windows

```
winpty bash
```

build

```
./build.sh
```

encrypt

```
./enc.sh /path/to/source/folder /path/to/destination/folder
```

decrypt

```
./dec.sh ./dest/20220101_000000.dat /path/to/destination/folder
```

decrypt (not decompress)

```
./dec-tar.sh ./dest/20220101_000000.dat /path/to/destination/folder
```
